## 📋 Sobre o Projeto

A configuração foi desenvolvida para atender aos requisitos da atividade de infraestrutura:
- Um banco de dados **NoSQL (MongoDB)**
- Um banco de dados **Relacional (MySQL)**
- Comunicação entre os containers através de uma **Network Personalizada**
- Persistência de dados utilizando **Volumes**
